性能实测#

通过一系列基准测试来评估苏黎世 ZRH1 机房这台 VPS 的实际性能表现。

root@r1089906:~# sysbench cpu --threads=1 run
CPU speed:
    events per second:  1504.47
Latency (ms):
         min:                                    0.60
         avg:                                    0.66
         95th percentile:                        0.77

单核每秒事件处理数达到 1504.47，平均延迟 0.66 毫秒。这个成绩在 AMD EPYC 7662 单核心中属于正常水平，应对日常网站运行、脚本处理等任务没什么压力。相比同价位的其他 VPS，这个 CPU 性能处于中上档次。

root@r1089906:~# sysbench memory run

磁盘 IO 性能测试#

使用 fio 工具测试 4K 随机读写和顺序读写性能。

4K 随机读写测试:

read: IOPS=7624， BW=29.8MiB/s
  clat percentiles (usec):
     | 50.00th=[   69]， 95.00th=[  167]， 99.00th=[  388]
     
write: IOPS=7612， BW=29.7MiB/s
  clat percentiles (nsec):
     | 50.00th=[  3376]， 95.00th=[  9280]， 99.00th=[ 21120]

4K 随机读取 IOPS 达到 7624，写入 IOPS 为 7612，读写带宽都在 30MB/s 左右。这个成绩确实达到了 NVMe 级别的性能表现，读取延迟中位数 69 微秒，95% 请求在 167 微秒内完成，写入延迟更是低至纳秒级别，响应速度非常快。

顺序读写测试:

read: IOPS=756， BW=756MiB/s
write: IOPS=804， BW=805MiB/s

顺序读取速度 756 MB/s，写入速度 805 MB/s，接近 1GB/s 的传输速率。这个数据证明磁盘确实是真实的 NVMe 固态硬盘，而不是 SATA SSD。对于需要处理大文件、数据库操作、编译任务等场景，这样的磁盘性能能明显提升效率。

值得一提的是，测试过程中磁盘利用率达到 98%，说明 IO 性能没有被严格限制，可以充分发挥硬件能力。

从数据来看，香港节点延迟最低为 169ms，国内三大运营商中移动网络延迟相对较低(平均 182-343ms)，电信次之(平均 226-313ms)，联通稍高(平均 228-333ms)。沿海发达地区延迟普遍低于西部偏远地区，这符合网络基础设施分布规律。

丢包率测试#

全国范围内综合丢包率为 6.85%，这个丢包率偏高，已经会影响正常使用体验。6-7% 的丢包对于网页浏览、文件传输等场景勉强可用，但对于视频会议、游戏、实时通讯等对网络稳定性要求高的应用会有明显影响。

路由追踪测试#

从上海出发的数据包先经过美国加利福尼亚州圣何塞，再到达瑞士苏黎世。这条绕道美国的路由明显不是最优路径，直连欧洲的路由应该经过香港或新加坡直达。绕美线路会增加额外的 100-150ms 延迟，这也是回国延迟偏高的主要原因。

这种情况在晚高峰时段可能更严重，因为国际出口带宽拥堵时运营商会选择更曲折的路由。

实际下载速度测试#

使用 iperf3 从国内家庭宽带(100Mbps 电信)测试从 VPS 下载的实际速度:

10 秒内传输 3.19 MB，平均速度 2.68 Mbps，折合约 335 KB/s。在 VPS 端已开启 BBR 拥塞控制的情况下，速度仍然只有 2-3 Mbps，远未达到家庭宽带的上限。测试过程中重传次数达到 403 次，说明丢包问题确实存在，影响了传输效率。

速度在测试过程中波动较大，从 315 Kbps 到 4.96 Mbps 不等，说明网络质量不够稳定。

IP 质量与流媒体解锁测试#

IP 质量检测#

通过 IPQuality 脚本对 IP 地址进行全面检测，主要结果如下:

  • 基础信息: IP 归属显示为德国，但实际 VPS 位于瑞士苏黎世，不同数据库对地理位置判断存在差异
  • IP 类型: 所有数据库均识别为机房 IP，属于正常的数据中心 IP
  • 风险评分: IP2Location 和 IPQS 评分为高风险(80-100 分)，Scamalytics 评为中风险(22 分)，其他数据库评为低风险。
  • 风险因子: 多数数据库检测到代理和服务器特征，但未检测到 Tor、滥用、机器人等严重问题
  • 邮局连通性: 25 端口开放可用，大部分主流邮件服务商可正常通信，也有部分邮箱不可用
  • 黑名单状态: 439 个数据库中仅 48 个标记，无黑名单记录，整体信誉良好

这个 IP 质量属于标准机房 IP 水平，适合建站、跑服务等常规用途，但不适合需要住宅 IP 的特殊场景。

流媒体解锁测试#

CDN 节点显示为法兰克福，Netflix 和 YouTube 首选 CDN 均为法兰克福。整体来看，主流流媒体平台大部分可以解锁，Netflix 和 Amazon Prime 都能正常使用，但 Disney+ 和 Spotify 不可用。对于需要看 Netflix 的用户来说，这个节点基本满足需求。
